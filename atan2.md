# The Atan2 Function
## tags
#javascript #atan2 #tan #trigonometry 

## explanation
Functions the same way as an `atan()` function with some special exceptions. Its very useful for converting a x & y velocity into an angle of orientation. 

## examples
```javascript
Math.atan2(1, 1);
```
